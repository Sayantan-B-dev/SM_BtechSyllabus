### Topic Recap: CPU Subsystems, Control Unit, Instruction Set

**Key Concepts**:
- **Von Neumann Architecture**: Single shared memory for instructions and data.
- **CPU Components**: ALU (arithmetic/logic), Control Unit (fetch/decode/execute), Registers (fast storage inside CPU).
- **Memory Unit**: Stores instructions and data.
- **I/O Subsystems**: Handle input and output devices.
- **System Bus**: Connects CPU, memory, and I/O (data bus, address bus, control bus).
- **Control Unit**: Generates control signals to coordinate operations.
- **Instruction Set**: Collection of instructions a CPU can execute.

### Problem 2: Instruction Execution Cycle

Describe the steps for executing the instruction `ADD R1, R2` (add contents of R2 to R1).

**Solution**:

1. **Fetch**: Control unit sends the PC (Program Counter) value to the address bus. Memory returns the instruction at that address. PC is incremented.
2. **Decode**: Control unit decodes the binary instruction to determine it's `ADD R1, R2`.
3. **Read Operands**: Control unit enables R2 to send its value to the ALU input. R1's current value is sent to the other ALU input.
4. **Execute**: ALU performs addition. Result is placed on the internal data bus.
5. **Write Back**: Result is written back into register R1.
6. **Repeat**: Control unit fetches the next instruction from memory.

### Problem 4: CPU vs Microprocessor

Differentiate between CPU and Microprocessor.

| Feature | CPU | Microprocessor |
|---------|-----|----------------|
| Definition | Component that executes instructions | CPU fabricated on a single IC chip |
| Contains | ALU, CU, registers | All CPU components plus cache, memory controller on one chip |
| Example | Intel Core i7, AMD Ryzen | Same as CPU in modern context |
| Physical size | Can span multiple chips (older) | Single chip (modern) |

In modern usage, the terms are often used interchangeably because CPUs are manufactured as microprocessors.
