## Notes

### Topic Recap: SQL Advanced Queries (Subqueries, Joins, Aggregation)

**Joins**:
- `INNER JOIN`: Matching rows from both tables.
- `LEFT JOIN`: All rows from left, NULLs for non-matching right.
- `RIGHT JOIN`: All rows from right, NULLs for non-matching left.
- `FULL JOIN`: All rows from both tables.
- `CROSS JOIN`: Cartesian product.

**Subqueries**:
- `WHERE col IN (SELECT ...)`
- `WHERE col = (SELECT ...)` (single value)
- `WHERE EXISTS (SELECT ...)`
- `WHERE col > ALL/SOME/ANY (SELECT ...)`

**Aggregation**:
- `COUNT()`, `SUM()`, `AVG()`, `MAX()`, `MIN()`
- `GROUP BY`, `HAVING` (filter groups)

### Problem 1: JOIN Operations

Using Student, Course, Enrollment tables:
```sql
-- Inner join: Students with their enrolled courses
SELECT s.Name, c.Title, e.Grade
FROM Student s
INNER JOIN Enrollment e ON s.RollNo = e.RollNo
INNER JOIN Course c ON e.CourseCode = c.CourseCode;

-- Left join: All students, even those with no enrollments
SELECT s.Name, e.CourseCode, e.Grade
FROM Student s
LEFT JOIN Enrollment e ON s.RollNo = e.RollNo;

-- Students with no enrollments
SELECT s.Name
FROM Student s
LEFT JOIN Enrollment e ON s.RollNo = e.RollNo
WHERE e.RollNo IS NULL;
```

### Problem 2: Subqueries

```sql
-- Students enrolled in CS301
SELECT Name FROM Student
WHERE RollNo IN (
    SELECT RollNo FROM Enrollment WHERE CourseCode = 'CS301'
);

-- Students with GPA higher than average
SELECT Name, GPA FROM Student
WHERE GPA > (SELECT AVG(GPA) FROM Student);

-- Courses with no enrollments
SELECT Title FROM Course c
WHERE NOT EXISTS (
    SELECT 1 FROM Enrollment e WHERE e.CourseCode = c.CourseCode
);

-- Students enrolled in ALL CS courses
SELECT Name FROM Student s
WHERE NOT EXISTS (
    SELECT CourseCode FROM Course WHERE CourseCode LIKE 'CS%'
    EXCEPT
    SELECT CourseCode FROM Enrollment e WHERE e.RollNo = s.RollNo
);
```

### Problem 3: Aggregation with GROUP BY

```sql
-- Number of students per department
SELECT Dept, COUNT(*) AS StudentCount
FROM Student
GROUP BY Dept;

-- Average GPA per department
SELECT Dept, AVG(GPA) AS AvgGPA
FROM Student
GROUP BY Dept;

-- Departments with average GPA > 8.0
SELECT Dept, AVG(GPA) AS AvgGPA
FROM Student
GROUP BY Dept
HAVING AVG(GPA) > 8.0;

-- Maximum, minimum, and average GPA by department
SELECT Dept,
       COUNT(*) AS Count,
       MAX(GPA) AS MaxGPA,
       MIN(GPA) AS MinGPA,
       AVG(GPA) AS AvgGPA
FROM Student
GROUP BY Dept;
```

### Problem 4: Subqueries in SELECT and FROM

```sql
-- Subquery in SELECT: show GPA relative to department average
SELECT Name, Dept, GPA,
       (SELECT AVG(GPA) FROM Student s2 WHERE s2.Dept = s1.Dept) AS DeptAvgGPA
FROM Student s1;

-- Subquery in FROM: find departments exceeding overall average
SELECT Dept, AvgGPA
FROM (
    SELECT Dept, AVG(GPA) AS AvgGPA
    FROM Student
    GROUP BY Dept
) AS DeptStats
WHERE AvgGPA > (SELECT AVG(GPA) FROM Student);
```

### Problem 5: Complex Query

Find the names of students who have the highest GPA in their department.

```sql
SELECT s.Name, s.Dept, s.GPA
FROM Student s
WHERE s.GPA = (
    SELECT MAX(GPA)
    FROM Student
    WHERE Dept = s.Dept
)
ORDER BY s.Dept;
```

Alternative using window functions:
```sql
SELECT Name, Dept, GPA
FROM (
    SELECT Name, Dept, GPA,
           RANK() OVER (PARTITION BY Dept ORDER BY GPA DESC) AS rnk
    FROM Student
) ranked
WHERE rnk = 1;
```
