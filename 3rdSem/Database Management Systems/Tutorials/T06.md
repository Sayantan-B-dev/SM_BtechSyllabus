### Topic Recap: SQL Basics — DDL (CREATE, ALTER, DROP) and DML (INSERT, UPDATE, DELETE, SELECT)

**SQL DDL Commands**:
- `CREATE TABLE table_name (column1 datatype constraint, ...)`
- `ALTER TABLE table_name ADD/DROP/MODIFY column`
- `DROP TABLE table_name`
- `TRUNCATE TABLE table_name`

**SQL DML Commands**:
- `INSERT INTO table_name VALUES (v1, v2, ...)`
- `INSERT INTO table_name (col1, col2) VALUES (v1, v2)`
- `UPDATE table_name SET column = value WHERE condition`
- `DELETE FROM table_name WHERE condition`
- `SELECT col1, col2 FROM table_name WHERE condition`

### Problem 1: Creating Tables

Create a College database with tables Student, Course, and Enrollment.

```sql
CREATE DATABASE College;
USE College;

CREATE TABLE Student (
    RollNo INT PRIMARY KEY,
    Name VARCHAR(50) NOT NULL,
    Age INT CHECK (Age >= 16),
    Dept VARCHAR(10),
    GPA DECIMAL(3,2) CHECK (GPA >= 0 AND GPA <= 10)
);

CREATE TABLE Course (
    CourseCode VARCHAR(10) PRIMARY KEY,
    Title VARCHAR(100) NOT NULL,
    Credits INT CHECK (Credits > 0),
    Instructor VARCHAR(50)
);

CREATE TABLE Enrollment (
    RollNo INT,
    CourseCode VARCHAR(10),
    Semester VARCHAR(10),
    Grade CHAR(2),
    PRIMARY KEY (RollNo, CourseCode, Semester),
    FOREIGN KEY (RollNo) REFERENCES Student(RollNo),
    FOREIGN KEY (CourseCode) REFERENCES Course(CourseCode)
);
```

### Problem 3: SELECT with WHERE

Query examples using the College database.

```sql
-- All students
SELECT * FROM Student;

-- CS department students
SELECT * FROM Student WHERE Dept = 'CS';

-- Students with GPA >= 8.0
SELECT Name, GPA FROM Student WHERE GPA >= 8.0;

-- Students not in EE department
SELECT * FROM Student WHERE Dept <> 'EE';

-- Students with age between 19 and 21
SELECT * FROM Student WHERE Age BETWEEN 19 AND 21;
```

### Problem 4: UPDATE and DELETE

```sql
-- Update GPA for student 101
UPDATE Student SET GPA = 8.7 WHERE RollNo = 101;

-- Give all CS students a 0.5 GPA bonus
UPDATE Student SET GPA = GPA + 0.5 WHERE Dept = 'CS' AND GPA + 0.5 <= 10;

-- Delete enrollment for a specific student
DELETE FROM Enrollment WHERE RollNo = 104 AND CourseCode = 'ME101';

-- Delete all students from ME department
DELETE FROM Student WHERE Dept = 'ME';
-- Note: Check foreign key constraints first
```

### Problem 5: ORDER BY and LIMIT

```sql
-- Students sorted by GPA descending
SELECT * FROM Student ORDER BY GPA DESC;

-- Top 3 students by GPA
SELECT * FROM Student ORDER BY GPA DESC LIMIT 3;

-- Students sorted by department then by name
SELECT * FROM Student ORDER BY Dept, Name;

-- Youngest student
SELECT * FROM Student ORDER BY Age ASC LIMIT 1;
```
