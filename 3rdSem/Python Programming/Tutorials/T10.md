### Topic Recap: Function Types, Arguments, Scope

**Types of Arguments**:
- **Positional arguments**: matched by position.
- **Keyword arguments**: matched by parameter name.
- **Default arguments**: parameters with default values.
- **Variable-length arguments**:
  - `*args` — tuple of positional arguments
  - `**kwargs` — dictionary of keyword arguments

**Scope**:
- **Local**: variables defined inside a function.
- **Global**: variables defined at module level.
- `global` keyword to modify global variables inside a function.

**Lambda Functions**: Anonymous one-line functions: `lambda x: x**2`

### Problem 1: Variable-Length Arguments (*args)

Write a function to calculate the sum of any number of arguments.

```python
def sum_all(*args):
    total = 0
    for num in args:
        total += num
    return total

print("sum_all(1, 2, 3) = " + str(sum_all(1, 2, 3)))
print("sum_all(10, 20, 30, 40, 50) = " + str(sum_all(10, 20, 30, 40, 50)))
print("sum_all() = " + str(sum_all()))
```

**Output:**
```
sum_all(1, 2, 3) = 6
sum_all(10, 20, 30, 40, 50) = 150
sum_all() = 0
```

### Problem 2: Keyword Arguments (**kwargs)

Write a function to print student details using keyword arguments.

```python
def print_student(**kwargs):
    print("Student Details:")
    for key, value in kwargs.items():
        print("  " + key.capitalize() + ": " + str(value))

print_student(name="Alice", age=20, course="CS", grade="A")
print()
print_student(name="Bob", major="Mathematics")
```

**Output:**
```
Student Details:
  Name: Alice
  Age: 20
  Course: CS
  Grade: A

Student Details:
  Name: Bob
  Major: Mathematics
```

### Problem 3: Combining *args and **kwargs

Write a function that accepts both positional and keyword variable arguments.

```python
def display_info(title, *args, **kwargs):
    print("Title: " + title)
    print("Positional extras (" + str(len(args)) + "):")
    for i, arg in enumerate(args):
        print("  " + str(i + 1) + ". " + str(arg))
    print("Keyword extras (" + str(len(kwargs)) + "):")
    for key, value in kwargs.items():
        print("  " + key + ": " + str(value))

display_info("Report", 42, 99, author="Alice", date="2026-09-16")
```

**Output:**
```
Title: Report
Positional extras (2):
  1. 42
  2. 99
Keyword extras (2):
  author: Alice
  date: 2026-09-16
```

### Problem 4: Lambda Functions

Write a program to demonstrate lambda functions with `map`, `filter`, `sorted`.

```python
numbers = [5, 2, 8, 1, 9, 3, 7]

squared = list(map(lambda x: x**2, numbers))
print("Squared: " + str(squared))

evens = list(filter(lambda x: x % 2 == 0, numbers))
print("Evens: " + str(evens))

sorted_desc = sorted(numbers, key=lambda x: x, reverse=True)
print("Sorted desc: " + str(sorted_desc))

points = [(1, 5), (3, 2), (2, 4), (4, 1)]
sorted_by_y = sorted(points, key=lambda p: p[1])
print("Points sorted by y: " + str(sorted_by_y))
```

**Output:**
```
Squared: [25, 4, 64, 1, 81, 9, 49]
Evens: [2, 8]
Sorted desc: [9, 8, 7, 5, 3, 2, 1]
Points sorted by y: [(4, 1), (3, 2), (2, 4), (1, 5)]
```

### Problem 5: Global and Local Scope

Write a program to demonstrate variable scope.

```python
x = 10  # global

def modify_global():
    global x
    x = 20
    print("Inside function: x = " + str(x))

def local_var():
    x = 30
    print("Inside local_var: x = " + str(x))

print("Before: x = " + str(x))
modify_global()
print("After modify_global: x = " + str(x))
local_var()
print("After local_var: x = " + str(x))
```

**Output:**
```
Before: x = 10
Inside function: x = 20
After modify_global: x = 20
Inside local_var: x = 30
After local_var: x = 20
```
