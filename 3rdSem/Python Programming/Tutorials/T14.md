### Topic Recap: Classes, Objects, Constructors

**Object-Oriented Programming (OOP)**:
- **Class**: Blueprint for creating objects (instances).
- **Object**: Instance of a class.
- `__init__` method: Constructor called when an object is created.
- `self`: Reference to the current instance.
- **Instance variables**: Unique to each instance.
- **Class variables**: Shared across all instances.
- **Methods**: Functions defined inside a class.

**Special Methods (Dunder/Magic)**:
- `__str__`: String representation for print().
- `__repr__`: Unambiguous representation.
- `__len__`: Called by len().
- `__eq__`, `__lt__`, etc.: Comparison operators.

### Problem 1: Basic Class and Object

Write a class `Student` with instance variables and a method.

```python
class Student:
    def __init__(self, name, age, grade):
        self.name = name
        self.age = age
        self.grade = grade

    def display_info(self):
        return self.name + " (" + str(self.age) + ") - Grade: " + self.grade

s1 = Student("Alice", 20, "A")
s2 = Student("Bob", 22, "B")
print(s1.display_info())
print(s2.display_info())
```

**Output:**
```
Alice (20) - Grade: A
Bob (22) - Grade: B
```

### Problem 3: Bank Account Class

Write a class `BankAccount` with deposit, withdraw, and balance methods.

```python
class BankAccount:
    def __init__(self, account_holder, initial_balance=0):
        self.account_holder = account_holder
        self.balance = initial_balance
        self.transactions = []

    def deposit(self, amount):
        if amount > 0:
            self.balance += amount
            self.transactions.append("Deposited: $" + str(amount))
            return True
        return False

    def withdraw(self, amount):
        if 0 < amount <= self.balance:
            self.balance -= amount
            self.transactions.append("Withdrew: $" + str(amount))
            return True
        return False

    def get_balance(self):
        return self.balance

    def print_statement(self):
        print("Account statement for " + self.account_holder + ":")
        for t in self.transactions:
            print("  " + t)
        print("  Current balance: $" + str(self.balance))

acc = BankAccount("Alice", 1000)
acc.deposit(500)
acc.withdraw(200)
acc.deposit(300)
acc.print_statement()
```

**Output:**
```
Account statement for Alice:
  Deposited: $500
  Withdrew: $200
  Deposited: $300
  Current balance: $1600
```

### Problem 4: Special Methods (__str__ and __len__)

Write a class `Book` that implements special methods.

```python
class Book:
    def __init__(self, title, author, pages):
        self.title = title
        self.author = author
        self.pages = pages

    def __str__(self):
        return "'" + self.title + "' by " + self.author

    def __repr__(self):
        return "Book('" + self.title + "', '" + self.author + "', " + str(self.pages) + ")"

    def __len__(self):
        return self.pages

    def __eq__(self, other):
        if isinstance(other, Book):
            return self.title == other.title and self.author == other.author
        return False

b1 = Book("Python Basics", "John Smith", 300)
b2 = Book("Python Basics", "John Smith", 300)
b3 = Book("Advanced Python", "Jane Doe", 450)

print(str(b1))
print(repr(b1))
print("Pages: " + str(len(b1)))
print("b1 == b2: " + str(b1 == b2))
print("b1 == b3: " + str(b1 == b3))
```

**Output:**
```
'Python Basics' by John Smith
Book('Python Basics', 'John Smith', 300)
Pages: 300
b1 == b2: True
b1 == b3: False
```

### Problem 5: Library Management System

Write a simple library system with classes.

```python
class Book:
    def __init__(self, isbn, title, author):
        self.isbn = isbn
        self.title = title
        self.author = author
        self.is_borrowed = False

    def __str__(self):
        status = "Available" if not self.is_borrowed else "Borrowed"
        return self.title + " (" + status + ")"

class Library:
    def __init__(self, name):
        self.name = name
        self.books = []

    def add_book(self, book):
        self.books.append(book)

    def borrow_book(self, isbn):
        for book in self.books:
            if book.isbn == isbn and not book.is_borrowed:
                book.is_borrowed = True
                return True
        return False

    def return_book(self, isbn):
        for book in self.books:
            if book.isbn == isbn and book.is_borrowed:
                book.is_borrowed = False
                return True
        return False

    def display_books(self):
        print(self.name + " Catalog:")
        for book in self.books:
            print("  " + str(book))

lib = Library("City Library")
lib.add_book(Book("001", "Python 101", "Alice"))
lib.add_book(Book("002", "Data Structures", "Bob"))
lib.add_book(Book("003", "Algorithms", "Charlie"))

lib.display_books()
print("\nBorrowing '001'...")
lib.borrow_book("001")
lib.display_books()
print("\nReturning '001'...")
lib.return_book("001")
lib.display_books()
```

**Output:**
```
City Library Catalog:
  Python 101 (Available)
  Data Structures (Available)
  Algorithms (Available)

Borrowing '001'...
City Library Catalog:
  Python 101 (Borrowed)
  Data Structures (Available)
  Algorithms (Available)

Returning '001'...
City Library Catalog:
  Python 101 (Available)
  Data Structures (Available)
  Algorithms (Available)
```
