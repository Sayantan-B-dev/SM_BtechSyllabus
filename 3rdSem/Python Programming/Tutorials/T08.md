### Topic Recap: Dictionaries

**Dictionaries**:
- Unordered, mutable collection of key-value pairs.
- Created with curly braces: `d = {"key": "value"}`
- Keys must be immutable (strings, numbers, tuples).
- Values can be any data type.
- Python 3.7+: dictionaries maintain insertion order.

**Common Dictionary Methods**:
- `keys()`, `values()`, `items()` — view objects
- `get(key, default)` — returns value or default if key missing
- `pop(key, default)` — removes and returns value
- `popitem()` — removes and returns last inserted (key, value)
- `update(dict2)` — merges another dictionary
- `setdefault(key, default)` — returns value if exists, else inserts default
- `copy()`, `clear()`
- `fromkeys(seq, value)` — creates dict from sequence

### Problem 1: Creating and Accessing Dictionaries

Write a program to create a dictionary and access its elements.

```python
student = {
    "name": "Alice",
    "age": 20,
    "course": "Computer Science",
    "marks": 88
}
print("Student dictionary: " + str(student))
print("Name: " + student["name"])
print("Age: " + str(student.get("age")))
print("Phone (with default): " + str(student.get("phone", "Not available")))
```

**Output:**
```
Student dictionary: {'name': 'Alice', 'age': 20, 'course': 'Computer Science', 'marks': 88}
Name: Alice
Age: 20
Phone (with default): Not available
```

### Problem 2: Iterating Over a Dictionary

Write a program to iterate over keys, values, and items.

```python
student = {"name": "Bob", "age": 22, "city": "Delhi", "grade": "A"}

print("Keys:")
for key in student.keys():
    print("  " + key)

print("Values:")
for value in student.values():
    print("  " + str(value))

print("Items:")
for key, value in student.items():
    print("  " + key + ": " + str(value))
```

**Output:**
```
Keys:
  name
  age
  city
  grade
Values:
  Bob
  22
  Delhi
  A
Items:
  name: Bob
  age: 22
  city: Delhi
  grade: A
```

### Problem 3: Word Frequency Counter

Write a program to count word frequencies in a sentence.

```python
sentence = input("Enter a sentence: ")
words = sentence.lower().split()
freq = {}
for word in words:
    clean_word = word.strip(".,!?")
    freq[clean_word] = freq.get(clean_word, 0) + 1
for word, count in freq.items():
    print("'" + word + "': " + str(count))
```

**Output:**
```
Enter a sentence: The quick brown fox jumps over the lazy dog the quick fox
'the': 2
'quick': 2
'brown': 1
'fox': 2
'jumps': 1
'over': 1
'lazy': 1
'dog': 1
```

### Problem 5: Nested Dictionary

Write a program to manage a nested dictionary of employees.

```python
employees = {
    "E001": {"name": "Alice", "dept": "IT", "salary": 60000},
    "E002": {"name": "Bob", "dept": "HR", "salary": 50000},
    "E003": {"name": "Charlie", "dept": "IT", "salary": 70000}
}
print("Employee Details:")
for emp_id, details in employees.items():
    print("  " + emp_id + ": " + details["name"] + ", " + details["dept"] + ", $" + str(details["salary"]))

total_salary = sum(emp["salary"] for emp in employees.values())
print("\nTotal salary expenditure: $" + str(total_salary))
```

**Output:**
```
Employee Details:
  E001: Alice, IT, $60000
  E002: Bob, HR, $50000
  E003: Charlie, IT, $70000

Total salary expenditure: $180000
```

### Problem 6: Merging Dictionaries

Write a program to merge two dictionaries.

```python
dict1 = {"a": 1, "b": 2, "c": 3}
dict2 = {"b": 20, "d": 4, "e": 5}

merged = {**dict1, **dict2}
print("dict1: " + str(dict1))
print("dict2: " + str(dict2))
print("Merged (dict2 overwrites dict1): " + str(merged))

dict1.update(dict2)
print("After update: " + str(dict1))
```

**Output:**
```
dict1: {'a': 1, 'b': 2, 'c': 3}
dict2: {'b': 20, 'd': 4, 'e': 5}
Merged (dict2 overwrites dict1): {'a': 1, 'b': 20, 'c': 3, 'd': 4, 'e': 5}
After update: {'a': 1, 'b': 20, 'c': 3, 'd': 4, 'e': 5}
```
