### Topic Recap: List Operations, Functions, and Methods

**List Methods**:
- `append(x)` — adds x at end
- `extend(iterable)` — adds all elements from iterable
- `insert(i, x)` — inserts x at index i
- `remove(x)` — removes first occurrence of x
- `pop(i)` — removes and returns element at i (default last)
- `index(x)` — returns index of first occurrence
- `count(x)` — counts occurrences of x
- `sort()` — sorts in place
- `reverse()` — reverses in place
- `copy()` — returns a shallow copy
- `clear()` — removes all elements

**List Comprehension**: `[expression for item in iterable if condition]`

### Problem 1: List Methods Demonstration

Write a program to demonstrate various list methods.

```python
nums = [3, 1, 4, 1, 5, 9, 2, 6, 5]
print("Original: " + str(nums))

nums.sort()
print("Sorted: " + str(nums))

nums.reverse()
print("Reversed: " + str(nums))

print("Count of 5: " + str(nums.count(5)))
print("Index of 9: " + str(nums.index(9)))

nums.pop()
print("After pop: " + str(nums))

nums.extend([7, 8])
print("After extend: " + str(nums))
```

**Output:**
```
Original: [3, 1, 4, 1, 5, 9, 2, 6, 5]
Sorted: [1, 1, 2, 3, 4, 5, 5, 6, 9]
Reversed: [9, 6, 5, 5, 4, 3, 2, 1, 1]
Count of 5: 2
Index of 9: 0
After pop: [9, 6, 5, 5, 4, 3, 2, 1]
After extend: [9, 6, 5, 5, 4, 3, 2, 1, 7, 8]
```

### Problem 2: List Comprehension — Squares

Write a program to generate a list of squares using list comprehension.

```python
n = int(input("Enter N: "))
squares = [i**2 for i in range(1, n + 1)]
print("Squares: " + str(squares))

even_squares = [i**2 for i in range(1, n + 1) if i % 2 == 0]
print("Even squares: " + str(even_squares))
```

**Output:**
```
Enter N: 10
Squares: [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
Even squares: [4, 16, 36, 64, 100]
```

### Problem 3: Filter Even and Odd

Write a program to separate even and odd numbers from a list.

```python
numbers = [12, 7, 5, 18, 23, 30, 15, 42]
even = [x for x in numbers if x % 2 == 0]
odd = [x for x in numbers if x % 2 != 0]
print("Numbers: " + str(numbers))
print("Even: " + str(even))
print("Odd: " + str(odd))
```

**Output:**
```
Numbers: [12, 7, 5, 18, 23, 30, 15, 42]
Even: [12, 18, 30, 42]
Odd: [7, 5, 23, 15]
```

### Problem 4: Matrix Using Nested Lists

Write a program to create and manipulate a 3x3 matrix.

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print("Matrix:")
for row in matrix:
    print(" ".join(str(x) for x in row))

transpose = [[row[i] for row in matrix] for i in range(3)]
print("\nTranspose:")
for row in transpose:
    print(" ".join(str(x) for x in row))
```

**Output:**
```
Matrix:
1 2 3
4 5 6
7 8 9

Transpose:
1 4 7
2 5 8
3 6 9
```

### Problem 5: List as Stack and Queue

Write a program to demonstrate stack (LIFO) and queue (FIFO) using lists.

```python
stack = []
stack.append(10)
stack.append(20)
stack.append(30)
print("Stack after pushes: " + str(stack))
print("Popped: " + str(stack.pop()))
print("Stack after pop: " + str(stack))
print()

queue = []
queue.append(10)
queue.append(20)
queue.append(30)
print("Queue after enqueues: " + str(queue))
print("Dequeued: " + str(queue.pop(0)))
print("Queue after dequeue: " + str(queue))
```

**Output:**
```
Stack after pushes: [10, 20, 30]
Popped: 30
Stack after pop: [10, 20]

Queue after enqueues: [10, 20, 30]
Dequeued: 10
Queue after dequeue: [20, 30]
```
