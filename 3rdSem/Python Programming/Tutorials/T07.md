### Topic Recap: Tuples

**Tuples**:
- Immutable ordered sequence of elements.
- Created with parentheses: `tup = (1, 2, 3)` or without: `tup = 1, 2, 3`
- Single element tuple: `tup = (1,)` (comma is required).
- Supports indexing, slicing, concatenation, repetition.
- Immutable — cannot modify elements after creation.
- Use cases: returning multiple values from functions, storing fixed data.

**Tuple Methods**:
- `count(x)` — counts occurrences of x
- `index(x)` — returns index of first occurrence

**Tuple Packing and Unpacking**:
- Packing: `tup = 1, 2, 3`
- Unpacking: `a, b, c = tup`

### Problem 1: Tuple Basics

Write a program to create and access tuples.

```python
tup1 = (10, 20, 30, 40, 50)
tup2 = "apple", "banana", "cherry"
tup3 = (1,)  # single element tuple

print("tup1: " + str(tup1))
print("tup2: " + str(tup2))
print("tup3: " + str(tup3))
print("First element: " + str(tup1[0]))
print("Last element: " + str(tup1[-1]))
print("Slice (1:4): " + str(tup1[1:4]))
```

**Output:**
```
tup1: (10, 20, 30, 40, 50)
tup2: ('apple', 'banana', 'cherry')
tup3: (1,)
First element: 10
Last element: 50
Slice (1:4): (20, 30, 40)
```

### Problem 2: Tuple Immutability

Write a program to demonstrate that tuples are immutable.

```python
tup = (1, 2, 3)
print("Original tuple: " + str(tup))
try:
    tup[0] = 10
except TypeError as e:
    print("Error: " + str(e))
```

**Output:**
```
Original tuple: (1, 2, 3)
Error: 'tuple' object does not support item assignment
```

### Problem 3: Tuple Packing and Unpacking

Write a program to demonstrate packing and unpacking.

```python
coordinates = 10, 20, 30
x, y, z = coordinates
print("Packed tuple: " + str(coordinates))
print("Unpacked: x=" + str(x) + ", y=" + str(y) + ", z=" + str(z))

a, b, *rest = (1, 2, 3, 4, 5)
print("a=" + str(a) + ", b=" + str(b) + ", rest=" + str(rest))
```

**Output:**
```
Packed tuple: (10, 20, 30)
Unpacked: x=10, y=20, z=30
a=1, b=2, rest=[3, 4, 5]
```

### Problem 4: Swapping Using Tuples

Write a program to swap variables using tuple packing/unpacking.

```python
a = 5
b = 10
print("Before: a=" + str(a) + ", b=" + str(b))
a, b = b, a
print("After: a=" + str(a) + ", b=" + str(b))
```

**Output:**
```
Before: a=5, b=10
After: a=10, b=5
```

### Problem 5: Returning Multiple Values from a Function

Write a function that returns min and max from a tuple.

```python
def find_min_max(numbers):
    return min(numbers), max(numbers)

data = (45, 12, 78, 34, 90, 23)
minimum, maximum = find_min_max(data)
print("Data: " + str(data))
print("Minimum: " + str(minimum))
print("Maximum: " + str(maximum))
```

**Output:**
```
Data: (45, 12, 78, 34, 90, 23)
Minimum: 12
Maximum: 90
```

### Problem 6: List of Tuples

Write a program to store student records as a list of tuples.

```python
students = [
    ("Alice", 85, "A"),
    ("Bob", 72, "B"),
    ("Charlie", 90, "A"),
    ("Diana", 60, "C")
]
print("Student Records:")
for name, marks, grade in students:
    print(name + ": " + str(marks) + " -> " + grade)

top_student = max(students, key=lambda s: s[1])
print("\nTop student: " + top_student[0] + " with " + str(top_student[1]) + " marks")
```

**Output:**
```
Student Records:
Alice: 85 -> A
Bob: 72 -> B
Charlie: 90 -> A
Diana: 60 -> C

Top student: Charlie with 90 marks
```
