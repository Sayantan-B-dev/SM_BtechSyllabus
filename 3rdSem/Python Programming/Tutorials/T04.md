### Topic Recap: String Operations, Slicing, Methods

**Strings**:
- Immutable sequence of characters enclosed in single/double/triple quotes.
- Indexing: positive (0 to len-1), negative (-1 to -len).
- Slicing: `str[start:stop:step]`
- Concatenation: `+`, Repetition: `*`
- Membership: `in`, `not in`

**Common String Methods**:
- `upper()`, `lower()`, `capitalize()`, `title()`, `swapcase()`
- `strip()`, `lstrip()`, `rstrip()`
- `split(sep)`, `join(list)`
- `find(sub)`, `index(sub)`, `count(sub)`
- `replace(old, new)`
- `startswith(sub)`, `endswith(sub)`
- `isalpha()`, `isdigit()`, `isalnum()`, `isspace()`

### Problem 1: String Reversal

Write a program to reverse a string using slicing.

```python
text = input("Enter a string: ")
reversed_text = text[::-1]
print("Reversed: " + reversed_text)
```

**Output:**
```
Enter a string: Python
Reversed: nohtyP
```

### Problem 2: Count Vowels in a String

Write a program to count vowels in a given string.

```python
text = input("Enter a string: ").lower()
vowels = "aeiou"
count = 0
for ch in text:
    if ch in vowels:
        count += 1
print("Number of vowels: " + str(count))
```

**Output:**
```
Enter a string: Hello World
Number of vowels: 3
```

### Problem 3: Palindrome Checker

Write a program to check if a string is a palindrome.

```python
text = input("Enter a string: ")
clean_text = text.replace(" ", "").lower()
if clean_text == clean_text[::-1]:
    print("\"" + text + "\" is a palindrome")
else:
    print("\"" + text + "\" is not a palindrome")
```

**Output:**
```
Enter a string: A man a plan a canal Panama
"A man a plan a canal Panama" is a palindrome
```

### Problem 5: Extract Email Domain

Write a program to extract the domain from an email address.

```python
email = input("Enter email: ")
at_index = email.find("@")
if at_index != -1:
    domain = email[at_index + 1:]
    print("Domain: " + domain)
else:
    print("Invalid email")
```

**Output:**
```
Enter email: user@example.com
Domain: example.com
```

### Problem 6: Character Frequency

Write a program to count the frequency of each character.

```python
text = input("Enter a string: ")
freq = {}
for ch in text:
    if ch in freq:
        freq[ch] += 1
    else:
        freq[ch] = 1
for ch, count in freq.items():
    print(repr(ch) + ": " + str(count))
```

**Output:**
```
Enter a string: hello
'h': 1
'e': 1
'l': 2
'o': 1
```
