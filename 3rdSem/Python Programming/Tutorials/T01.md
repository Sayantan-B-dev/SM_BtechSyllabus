### Topic Recap: Python Basics, Syntax, Variables, Data Types

A quick recap of fundamental concepts before solving problems:

- **Variables**: Containers for storing data values. No type declaration needed.
- **Data Types**: `int`, `float`, `str`, `bool`, `NoneType`
- **Basic Syntax**: Python uses indentation (4 spaces) instead of braces.
- **`print()`**: Used to display output.
- **`input()`**: Used to take user input (always returns a string).
- **Comments**: `#` for single line, `"""` for multi-line.

### Problem 1: Personal Greeting

Write a program that asks the user for their name and age, then prints a greeting.

```python
name = input("Enter your name: ")
age = input("Enter your age: ")
print("Hello " + name + "! You are " + age + " years old.")
```

**Output:**
```
Enter your name: Alice
Enter your age: 20
Hello Alice! You are 20 years old.
```

### Problem 2: Type Identification

Write a program that declares variables of different types and prints their types.

```python
a = 42
b = 3.14
c = "Python"

```

**Output:**
```

<class 'float'>
<class 'str'>
<class 'bool'>
<class 'NoneType'>
```

### Problem 3: Temperature Converter

Write a program to convert Celsius to Fahrenheit.

```python
celsius = float(input("Enter temperature in Celsius: "))
fahrenheit = (celsius * 9/5) + 32
print(str(celsius) + "C = " + str(fahrenheit) + "F")
```

**Output:**
```
Enter temperature in Celsius: 37
37.0C = 98.6F
```

### Problem 4: Simple Interest Calculator

Write a program to calculate simple interest.

```python
principal = float(input("Enter principal amount: "))
rate = float(input("Enter rate of interest: "))
time = float(input("Enter time in years: "))
si = (principal * rate * time) / 100
print("Simple Interest = " + str(si))
```

**Output:**
```
Enter principal amount: 10000
Enter rate of interest: 5
Enter time in years: 3
Simple Interest = 1500.0
```

### Problem 5: Swap Two Variables

Write a program to swap two numbers.

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))
print("Before swap: a = " + str(a) + ", b = " + str(b))
a, b = b, a
print("After swap: a = " + str(a) + ", b = " + str(b))
```

**Output:**
```
Enter first number: 10
Enter second number: 20
Before swap: a = 10, b = 20
After swap: a = 20, b = 10
```
