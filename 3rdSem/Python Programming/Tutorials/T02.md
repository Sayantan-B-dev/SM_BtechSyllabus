### Topic Recap: Operators, Decision Making (if/elif/else)

**Operators**:
- Arithmetic: `+`, `-`, `*`, `/`, `//`, `%`, `**`
- Comparison: `==`, `!=`, `<`, `>`, `<=`, `>=`
- Logical: `and`, `or`, `not`
- Assignment: `=`, `+=`, `-=`, `*=`, `/=`

**Decision Making**:
- `if condition:` — executes block if condition is True
- `if condition: else:` — one branch or the other
- `if condition: elif condition: else:` — multiple branches

### Problem 1: Even or Odd

Write a program to check if a number is even or odd.

```python
num = int(input("Enter a number: "))
if num % 2 == 0:
    print(str(num) + " is even")
else:
    print(str(num) + " is odd")
```

**Output:**
```
Enter a number: 7
7 is odd
```

### Problem 2: Largest of Three Numbers

Write a program to find the largest among three numbers.

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))
c = int(input("Enter third number: "))

if a >= b and a >= c:
    largest = a
elif b >= a and b >= c:
    largest = b
else:
    largest = c

print("Largest number is " + str(largest))
```

**Output:**
```
Enter first number: 45
Enter second number: 78
Enter third number: 23
Largest number is 78
```

### Problem 4: Leap Year Checker

Write a program to check if a year is a leap year.

```python
year = int(input("Enter a year: "))
if (year % 400 == 0) or (year % 4 == 0 and year % 100 != 0):
    print(str(year) + " is a leap year")
else:
    print(str(year) + " is not a leap year")
```

**Output:**
```
Enter a year: 2024
2024 is a leap year
```

### Problem 5: Arithmetic Calculator Using if-elif

Write a simple calculator for two numbers.

```python
a = float(input("Enter first number: "))
b = float(input("Enter second number: "))
op = input("Enter operator (+, -, *, /): ")

if op == "+":
    result = a + b
elif op == "-":
    result = a - b
elif op == "*":
    result = a * b
elif op == "/":
    if b != 0:
        result = a / b
    else:
        result = "Error: Division by zero"
else:
    result = "Invalid operator"

print("Result: " + str(result))
```

**Output:**
```
Enter first number: 15
Enter second number: 4
Enter operator (+, -, *, /): *
Result: 60.0
```
