### Topic Recap: Loops (for, while), break, continue

**Loops**:
- `for` loop: Iterates over a sequence (range, list, string).
  - `for i in range(n):` — 0 to n-1
  - `for i in range(start, stop, step):`
- `while` loop: Repeats while a condition is True.
- `break`: Exits the loop immediately.
- `continue`: Skips the rest of the current iteration and moves to the next.
- `pass`: Does nothing (placeholder).

### Problem 1: Sum of First N Natural Numbers

Write a program to calculate the sum of first N natural numbers.

```python
n = int(input("Enter N: "))
total = 0
for i in range(1, n + 1):
    total += i
print("Sum of first " + str(n) + " natural numbers = " + str(total))
```

**Output:**
```
Enter N: 10
Sum of first 10 natural numbers = 55
```

### Problem 3: Factorial Using While Loop

Write a program to find the factorial of a number.

```python
num = int(input("Enter a number: "))
fact = 1
i = 1
while i <= num:
    fact *= i
    i += 1
print("Factorial of " + str(num) + " = " + str(fact))
```

**Output:**
```
Enter a number: 6
Factorial of 6 = 720
```

### Problem 4: Break Statement — Find First Divisor

Write a program to find the first divisor of a number greater than 1.

```python
num = int(input("Enter a number: "))
for i in range(2, num):
    if num % i == 0:
        print(str(i) + " divides " + str(num))
        break
else:
    print(str(num) + " is prime")
```

**Output:**
```
Enter a number: 49
7 divides 49
```

### Problem 5: Continue Statement — Print Odd Numbers

Write a program to print all odd numbers from 1 to 20 using continue.

```python
for i in range(1, 21):
    if i % 2 == 0:
        continue
    print(i)
```

5

### Problem 6: Pattern Printing

Write a program to print a right-angled triangle pattern.

```python
n = int(input("Enter number of rows: "))
for i in range(1, n + 1):
    for j in range(i):
        print("*", end="")
    print()
```

**Output:**
```
Enter number of rows: 5
*
**
***
****
*****
```
