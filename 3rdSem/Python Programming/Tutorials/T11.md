### Topic Recap: Modules, Packages, Input-Output, File Handling

**Modules**: Python files (`.py`) containing functions and variables.
- Import: `import module_name`, `from module_name import function`
- Standard modules: `math`, `random`, `datetime`, `os`

**Packages**: Directories containing `__init__.py` and multiple modules.

**File Handling**:
- `open(filename, mode)` — modes: `'r'` (read), `'w'` (write), `'a'` (append), `'rb'`/`'wb'` (binary)
- `file.read()`, `file.readline()`, `file.readlines()`
- `file.write(text)`, `file.writelines(lines)`
- `with` statement for automatic file closing.

### Problem 1: Using the math Module

Write a program to use functions from the math module.

```python
import math

num = 16
print("Square root of " + str(num) + ": " + str(math.sqrt(num)))
print("Factorial of 7: " + str(math.factorial(7)))
print("Pi: " + str(math.pi))
print("E: " + str(math.e))
print("sin(90 degrees): " + str(math.sin(math.radians(90))))
print("ceil(4.2): " + str(math.ceil(4.2)))
print("floor(4.8): " + str(math.floor(4.8)))
```

**Output:**
```
Square root of 16: 4.0
Factorial of 7: 5040
Pi: 3.141592653589793
E: 2.718281828459045
sin(90 degrees): 1.0
ceil(4.2): 5
floor(4.8): 4
```

### Problem 2: Using the random Module

Write a program to generate random numbers and simulate a dice roll.

```python
import random

print("Random float (0-1): " + str(random.random()))
print("Random integer (1-100): " + str(random.randint(1, 100)))
print("Random choice from list: " + str(random.choice(["rock", "paper", "scissors"])))

dice1 = random.randint(1, 6)
dice2 = random.randint(1, 6)
print("Dice roll: " + str(dice1) + " + " + str(dice2) + " = " + str(dice1 + dice2))
```

**Output:**
```
Random float (0-1): 0.724859324789
Random integer (1-100): 57
Random choice from list: scissors
Dice roll: 3 + 6 = 9
```

### Problem 3: Writing to a File

Write a program to write user input to a file.

```python
text = input("Enter some text to save: ")
with open("sample.txt", "w") as file:
    file.write(text + "\n")
print("Text written to sample.txt")

with open("sample.txt", "a") as file:
    file.write("This is an appended line.\n")
print("Additional line appended.")
```

**Output:**
```
Enter some text to save: Hello, Python file handling!
Text written to sample.txt
Additional line appended.
```

### Problem 4: Reading from a File

Write a program to read and display file contents.

```python
with open("sample.txt", "r") as file:
    print("Full content:")
    content = file.read()
    print(content)

with open("sample.txt", "r") as file:
    print("Line by line:")
    for line in file:
        print("  Line: " + line.rstrip())
```

**Output:**
```
Full content:
Hello, Python file handling!
This is an appended line.

Line by line:
  Line: Hello, Python file handling!
  Line: This is an appended line.
```

### Problem 5: Copy a File

Write a program to copy the contents of one file to another.

```python
source = "sample.txt"
destination = "copy_sample.txt"

with open(source, "r") as src, open(destination, "w") as dest:
    for line in src:
        dest.write(line)

print("File copied from " + source + " to " + destination)

with open(destination, "r") as f:
    print("Contents of copy:\n" + f.read())
```

**Output:**
```
File copied from sample.txt to copy_sample.txt
Contents of copy:
Hello, Python file handling!
This is an appended line.
```

### Problem 6: Using datetime Module

Write a program to display current date and time.

```python
from datetime import datetime, timedelta

now = datetime.now()
print("Current date and time: " + str(now))
print("Formatted: " + now.strftime("%Y-%m-%d %H:%M:%S"))
print("Year: " + str(now.year))
print("Month: " + str(now.month))
print("Day: " + str(now.day))

tomorrow = now + timedelta(days=1)
print("Tomorrow: " + tomorrow.strftime("%Y-%m-%d"))
```

**Output:**
```
Current date and time: 2026-09-30 10:30:00.123456
Formatted: 2026-09-30 10:30:00
Year: 2026
Month: 9
Day: 30
Tomorrow: 2026-10-01
```
