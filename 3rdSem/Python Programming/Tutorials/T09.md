### Topic Recap: Dictionary Properties Wrap-Up, Introduction to Functions

**Dictionary Properties**:
- Membership test works on keys: `"key" in dict`
- `len(dict)` returns number of key-value pairs.
- Dictionary comprehension: `{k: v for k, v in iterable}`
- Dictionary is mutable — can add, modify, delete key-value pairs.

**Functions**:
- Defined with `def` keyword: `def function_name(parameters):`
- `return` statement sends back a value.
- Parameters can have default values.
- Functions can return multiple values as a tuple.
- `pass` can be used as a placeholder.

### Problem 1: Dictionary Comprehension

Write a program to create dictionaries using comprehension.

```python
squares = {x: x**2 for x in range(1, 11)}
print("Squares dictionary: " + str(squares))

even_squares = {x: x**2 for x in range(1, 11) if x % 2 == 0}
print("\nEven squares dictionary: " + str(even_squares))

word = "hello"
char_positions = {ch: [i for i, c in enumerate(word) if c == ch] for ch in set(word)}
print("\nCharacter positions in '" + word + "': " + str(char_positions))
```

**Output:**
```
Squares dictionary: {1: 1, 2: 4, 3: 9, 4: 16, 5: 25, 6: 36, 7: 49, 8: 64, 9: 81, 10: 100}

Even squares dictionary: {2: 4, 4: 16, 6: 36, 8: 64, 10: 100}

Character positions in 'hello': {'h': [0], 'e': [1], 'l': [2, 3], 'o': [4]}
```

### Problem 2: Basic Function

Write a function to check if a number is prime.

```python
def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

num = int(input("Enter a number: "))
if is_prime(num):
    print(str(num) + " is prime")
else:
    print(str(num) + " is not prime")
```

**Output:**
```
Enter a number: 17
17 is prime
```

### Problem 3: Function with Default Arguments

Write a function to calculate power with a default exponent.

```python
def power(base, exp=2):
    result = 1
    for _ in range(exp):
        result *= base
    return result

print("5^2 = " + str(power(5)))
print("5^3 = " + str(power(5, 3)))
print("2^10 = " + str(power(2, 10)))
```

**Output:**
```
5^2 = 25
5^3 = 125
2^10 = 1024
```

### Problem 4: Function Returning Multiple Values

Write a function to find min, max, and average of a list.

```python
def stats(numbers):
    return min(numbers), max(numbers), sum(numbers) / len(numbers)

data = [15, 22, 8, 35, 19, 42, 11]
minimum, maximum, avg = stats(data)
print("Data: " + str(data))
print("Min: " + str(minimum))
print("Max: " + str(maximum))
print("Average: " + str(avg))
```

**Output:**
```
Data: [15, 22, 8, 35, 19, 42, 11]
Min: 8
Max: 42
Average: 21.714285714285715
```

### Problem 5: Using a Dictionary to Dispatch Functions

Write a simple calculator using a function dictionary.

```python
def add(a, b): return a + b
def subtract(a, b): return a - b
def multiply(a, b): return a * b
def divide(a, b): return a / b if b != 0 else "Error"

operations = {"+": add, "-": subtract, "*": multiply, "/": divide}

a = float(input("Enter first number: "))
b = float(input("Enter second number: "))
op = input("Enter operator (+, -, *, /): ")

if op in operations:
    result = operations[op](a, b)
    print("Result: " + str(result))
else:
    print("Invalid operator")
```

**Output:**
```
Enter first number: 20
Enter second number: 5
Enter operator (+, -, *, /): /
Result: 4.0
```
